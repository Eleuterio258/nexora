# Inicio Realista

Para comecar um SaaS ERP, nao e necessario montar uma infraestrutura global.

## Base inicial recomendada

```text
1 VPS
Docker
Traefik
MySQL
Redis
Cloudflare
```

Essa base suporta com conforto algo como:

- 5.000 a 20.000 empresas

## Proximos passos possiveis

- arquitetura para 10 milhoes de usuarios
- criacao automatica de tenant por subdominio
- sistema de licencas SaaS e pagamentos
- infraestrutura barata com caminho claro de escala
