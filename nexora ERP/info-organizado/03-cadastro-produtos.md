# Cadastro de Produtos

Aqui ficam os produtos ou servicos que serao vendidos.

## Campos comuns

- ID do produto
- Nome do produto
- Descricao
- Preco
- IVA
- Quantidade em estoque
- Categoria
- Codigo do produto

## Exemplo de tabela

```text
produtos
---------
id
nome
descricao
preco
iva
estoque
categoria_id
created_at
```
