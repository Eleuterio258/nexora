# Cadastro de Clientes

Aqui ficam guardadas as informacoes do cliente que depois aparecem na fatura.

## Campos comuns

- ID do cliente
- Nome / Razao social
- NUIT
- Endereco
- Cidade
- Pais
- Telefone
- Email
- Data de criacao

## Exemplo de tabela

```text
clientes
---------
id
nome
nuit
endereco
telefone
email
created_at
```
