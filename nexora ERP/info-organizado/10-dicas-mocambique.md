# Dicas Importantes Para Mocambique

A fatura deve ter:

- Numero sequencial
- NUIT da empresa
- NUIT do cliente (se existir)
- IVA
- Data
- QR code (em alguns sistemas)

Tambem pode ser util aprofundar:

- Arquitetura completa de um sistema de faturacao profissional
- Estrutura de banco de dados completa para SaaS multi-tenant
- Design das telas do modulo de faturacao
